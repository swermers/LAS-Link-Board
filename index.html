Getting Started (macOS)

  1. Download VoiceType above — unzip and move VoiceType.app to your Applications or Documents folder
  2. Open for the first time — macOS blocks unsigned apps. Use one of these workarounds:
    • Right-click VoiceType.app → "Open" → click "Open" again
    • Terminal: xattr -cr ~/Applications/VoiceType.app
    • System Settings > Privacy & Security → click "Open Anyway"
  3. Allow microphone access when prompted
  4. Sign in with your LinkBoard account
  5. Use the floating mic or hotkey: Ctrl/Cmd + Shift + Space

Windows Setup

  1. Download the Windows version above
  2. Windows SmartScreen — click "More info" then "Run anyway"
  3. Allow microphone when prompted, sign in, and start using the mic button or hotkey
